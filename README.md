# WakeupApplication
## About
AndroidでDozeモードが開始されないようScreenを常にOnにしたり、GCMで叩き起こしたりするサンプルです。

## Environment
### Receiver(Android)
* Android Studio 2.0
* Kotlin 1.0.1-2
* TargetSdkVersion 23
* MinSdkVersion 21

### Sender(Ruby)
* Windows10(64bit)
* Ruby				2.3.0
* GCM (Gem)			0.1.1
